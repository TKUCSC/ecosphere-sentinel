## Getting Started

### Installing

**Step 1** Make sure you have Go version 1.3 installed.

**Step 2** Clone the repository & cd into it

```
>$ git clone git@github.com:TKUCSC/ecosphere-sentinel.git
```

### Compiling

**Step 1** Run the build script

```
>$ ./build.sh build
```

### Running

**Step 1**

```
>$ ./build.sh run
```
